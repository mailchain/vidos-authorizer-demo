# Vidos Managed Instance

The Vidos Managed Instance is a pre-configured Vidos Authorizer instance provided for demonstration and testing purposes. It allows you to try out the Vidos Authorizer Demo application without any setup required.

## What is the Managed Instance?

The Vidos Managed Instance is a Vidos Authorizer instance that:

- **Pre-configured and ready to use** - No configuration or setup steps required
- **Publicly accessible** - Available for anyone to test and experiment with
- **Maintained by Vidos** - Regularly updated and monitored for reliability
- **Ideal for demos and testing** - Perfect for exploring OID4VP credential verification flows

## Using the Managed Instance

The managed instance is selected by default when you first launch the application. Simply:

1. Select "Vidos Managed instance" in the Instance Type options (pre-selected by default)
2. Start creating authorization requests immediately
3. No additional configuration needed

## When to Use Your Own Instance

While the managed instance is great for getting started, you may want to use your own instance when:

- **Production use** - Running the application in a production environment
- **Custom configuration** - You need specific policies or custom settings
- **Privacy requirements** - You need full control over data and configurations
- **Integration testing** - Testing with your own infrastructure

To use your own instance:

1. Select "Own instance" in the Instance Type options
2. Follow the [Vidos Gateway Setup Guide](GATEWAY_SETUP.md) to create your own Vidos Authorizer and Gateway
3. Enter your Gateway authorizer URL in the provided input field
